IMAGE CLASSIFICATION USING PYTHON

Description

In this workshop, Students will learn the basics of image classification using TensorFlow and Python. The students build their own image classifier to classify different types of flowers. They will also learn the essential steps in machine learning such as working with training and testing datasets and how to use it to maximize learning.

Requirement

Software Required to download prior to Workshop: TensorFlow 1.7 and the latest version of Python.
